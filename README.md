# Task04_AI_ShaizaShahid_CoreTech

This project implements a Student Marks Prediction Model using machine learning.
The model is developed using Python and Linear Regression.

Tools and Libraries:
- Google Colab
- Python
- Pandas
- Scikit-learn
- Matplotlib

The repository includes:
- Google Colab notebook (.ipynb)
- Dataset (.csv)
- Project report
- Reflection section
- Screenshots of code, output, and graph

This project demonstrates basic artificial intelligence concepts such as data handling, model training, prediction, and visualization.
